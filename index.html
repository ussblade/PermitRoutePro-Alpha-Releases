    Route review before navigation

    The alpha flow is centered on permit capture, OCR processing, route verification, and waypoint-by-waypoint review.

    App use

    Basic workflow

    1. Add permit pages. Capture photos or select permit images from the device.
    2. Review OCR results. Confirm the extracted origin, destination, route segments, and restrictions.
    3. Verify the route. Compare the displayed path with the permit and tap each waypoint after review.
    4. Accept the route. Store the accepted route and audit record before starting the trip.
    5. Resume navigation. Reopen the verified trip and follow the permit-approved route.

    Android sideloading

    Install the APK

    1. Download the APK. Use the download button on this page from the Android device.
    2. Open the downloaded file. Android may ask which app should install the package.
    3. Allow installs from this source. If prompted, enable the permission for the browser or file manager you used.
    4. Finish installation. Return to the installer and tap Install.
    5. Open Permit Route Pro. Sign in, verify email if prompted, and start with a test permit.

    Alpha builds are distributed outside the Play Store. Only install APKs from this page or the linked public GitHub release repository.
